### Version Numbers

Version numbers should be in the format specified below. Supported release versions are alpha and beta. For shorthand you can use a and b respectively.

1.0

1.1alpha

1.1-alpha2

1.1-alpha-3

1.1.1

1.0.2beta

1.0.2-beta2

1.0.2-beta-3
